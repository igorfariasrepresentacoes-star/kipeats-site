Política de Privacidade — KIP Eats

Versão: privacy_v2026_04_02 · Última atualização: 02/04/2026

1. Introdução

Esta Política de Privacidade descreve como o aplicativo KIP Eats coleta, utiliza, armazena e protege os dados pessoais dos usuários, em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

2. Dados coletados

Dados fornecidos pelo usuário:

  • Nome
  • Telefone
  • Endereço de entrega

Dados de uso:

  • Histórico de pedidos
  • Preferências de consumo

Dados técnicos:

  • Endereço IP
  • Informações do dispositivo
  • Dados de acesso

3. Finalidade do tratamento

  • Processar pedidos
  • Viabilizar entregas
  • Comunicação com o usuário
  • Envio de atualizações de pedidos (incluindo WhatsApp)
  • Melhorar a experiência do usuário
  • Cumprir obrigações legais

4. Compartilhamento de dados

Os dados podem ser compartilhados com:

  • Estabelecimentos parceiros
  • Serviços de entrega
  • Plataformas tecnológicas (ex.: banco de dados e infraestrutura)
  • Autoridades legais, quando exigido

Não realizamos venda de dados pessoais.

5. Armazenamento e segurança

Os dados são armazenados em ambientes seguros, com medidas técnicas e organizacionais adequadas. Apesar disso, nenhum sistema é totalmente seguro.

6. Direitos do usuário

  • Solicitar acesso aos dados
  • Corrigir dados
  • Solicitar exclusão (também disponível direto no app, em Perfil > Excluir conta)
  • Revogar consentimento

Solicitações podem ser feitas através dos canais de contato.

7. Retenção de dados

Os dados são armazenados apenas pelo tempo necessário para:

  • Cumprir as finalidades
  • Atender obrigações legais
  • Resolver disputas

8. Cookies e tecnologias

O aplicativo pode utilizar tecnologias para melhorar a experiência, como armazenamento local (ex.: persistência do carrinho).

9. Alterações

Esta política pode ser atualizada a qualquer momento. A versão atual será sempre publicada nesta URL.
